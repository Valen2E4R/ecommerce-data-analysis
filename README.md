# E-commerce Data Analysis / dashboard incluido

## Live Demo
👉 https://ecommerce-data-analysis-uw7ajqqgxzlzplrrqjbkph.streamlit.app/

## 📊 Objetivo
Analizar datos de ventas para identificar patrones de compra, productos clave y oportunidades de negocio.

## 🛠️ Herramientas
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Análisis realizado
- Productos más vendidos
- Ingresos por país
- Clientes más valiosos
- Evolución temporal de ventas

## 📈 Insights clave
- Un pequeño grupo de productos concentra la mayoría de las ventas
- Algunos países generan significativamente más ingresos
- Existen clientes de alto valor que representan gran parte del revenue
- Identificación de clientes de alto valor
- Productos con mayor impacto en revenue
- Variación temporal de ventas

## 📊 Dashboard Features
- Filtros por país y fecha
- KPIs: Revenue, Orders, AOV
- Top productos y clientes
- Análisis temporal de ventas

## 🚀 Conclusión
El análisis permite identificar oportunidades de optimización en productos, mercados y segmentación de clientes.
